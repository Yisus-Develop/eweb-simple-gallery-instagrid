Instagram-style gallery plugin with custom post type and shortcodes.

== Description ==
Includes:
- Custom post type for gallery entries
- Feed shortcode and simple gallery shortcode
- Frontend modal/lightbox behavior
- Admin tools for ordering and image management

== Installation ==
1. Upload the plugin folder to /wp-content/plugins/.
2. Activate the plugin.
3. Use [instagrid_feed] or [simple_gallery_instagrid].
